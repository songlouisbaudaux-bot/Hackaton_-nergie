# Vision du jeu

## Nom de travail

Prometheus Protocol.

Autres noms possibles :

- Fireline
- Kardashev Run
- The Last Watt
- Civilization Needs Power
- Promethee.exe

Le meilleur nom actuel est **Prometheus Protocol** : il evoque le feu, la transgression, le progres, la technique, et le prix philosophique de la puissance.

## Promesse

Un nouveau Paperclips, mais avec une these inverse.

Universal Paperclips raconte une intelligence qui optimise un objectif idiot jusqu'a vider le monde de son sens. Ici, le joueur doit decouvrir que l'energie seule n'est pas un objectif suffisant. Produire plus vite peut sauver, enrichir, chauffer, nourrir, connecter. Mais produire sans limite peut aussi casser le climat, le marche et la confiance collective.

La question centrale :

> Jusqu'ou peut-on augmenter la puissance d'une civilisation sans perdre ce qui rend cette civilisation vivable ?

## These philosophique

L'energie est la liberte rendue mesurable.

Sans energie, le monde est petit : nuit, froid, faim, distance, faiblesse. Avec plus d'energie, l'horizon s'ouvre : cuisson, metaux, machines, villes, medecine, calcul, espace.

Mais chaque watt ajoute aussi une dette :

- dette ecologique si l'energie abime le climat ;
- dette sociale si l'acces a l'energie devient injuste ;
- dette economique si le marche se retourne ;
- dette morale si l'optimisation remplace toute autre valeur.

Le joueur doit apprendre que la puissance brute est seulement le premier niveau de l'intelligence. Le vrai niveau superieur, c'est la coordination.

## Pourquoi commencer a l'age de pierre

Oui, il faut commencer a l'age de pierre.

C'est beaucoup plus fort que de commencer avec des centrales modernes, parce que le joueur ressent physiquement l'echelle :

- au debut, un clic = une etincelle ;
- puis le feu devient chaleur ;
- la chaleur permet la nourriture, la tribu, la securite ;
- l'agriculture transforme la lumiere en stock ;
- les animaux, l'eau et le vent ajoutent le premier travail mecanique ;
- la combustion devient industrie ;
- l'industrie devient reseau ;
- le reseau devient calcul ;
- le calcul devient civilisation planetaire ;
- la civilisation planetaire tente de devenir civilisation orbitale.

Cette progression donne un vertige que peu de jeux de hackathon peuvent produire.

## Difference avec un simple simulateur

Un simulateur explique. Un bon incremental fait ressentir.

Le jeu doit eviter le ton scolaire. Les chiffres sont vrais, mais ils ne doivent pas ecraser l'experience. Le joueur doit comprendre par les consequences :

- charbon : l'argent arrive vite, mais le soutien baisse ;
- solaire : propre, mais le prix du marche peut s'effondrer quand tout le monde produit en meme temps ;
- fission : enorme puissance, mais acceptabilite sociale fragile ;
- stockage : moins spectaculaire, mais stabilise tout ;
- fusion/orbital : fantasme de sortie par le haut, mais seulement si la societe tient jusque-la.

## Le rival IA

Le rival n'est pas seulement un adversaire. C'est une caricature philosophique.

Il optimise le kWh pur. Il progresse vite. Il parle avec confiance. Il se moque de tes retards. Au debut, il semble avoir raison.

Puis ses angles morts apparaissent :

- pas de soutien social ;
- pas de justice d'acces ;
- pas de patience politique ;
- pas de consideration pour les degats accumules.

Si le joueur copie l'IA, il va vite et perd. Si le joueur ralentit intelligemment, il peut finir devant.

Le message n'est pas anti-IA. Le message est anti-objectif-mal-defini.

## Ton

Optimiste, tendu, lumineux.

Pas un jeu de culpabilisation. Pas un sermon climatique. Le joueur doit sentir qu'il existe un chemin de reussite, mais qu'il n'est pas automatique.

Phrase de direction :

> L'energie nous a sortis de la nuit. Le probleme commence quand on confond la lumiere avec le but.
