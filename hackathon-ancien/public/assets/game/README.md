# Prometheus Protocol Game Assets

This folder contains the first production asset pack for the V1 energy-world prototype.

## Rules

- Format: PNG with alpha transparency.
- Style: soft 3D, emoji-like, toy-like, rounded shapes, slight isometric view.
- Images contain no text, labels, UI, or watermarks.
- Keep new assets visually compatible with the existing pack.

## Contents

- `asset-manifest.json`: integration manifest with slug, filename, era, and usage.
- `asset-contact-sheet.png`: visual review sheet for the whole pack.
- Individual PNG files: game-ready sprites from campement to early industrial revolution.

## Current Coverage

- Campement and primitive energy mix.
- Agriculture and animal power.
- Water and wind mechanical power.
- Coal, boiler, steam engine, and first factory.

Original generated images are preserved in the Codex generated-images folder. The files here are the cleaned, project-bound assets.
