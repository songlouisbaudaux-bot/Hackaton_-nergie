# Pacing 10 minutes vers 2026

## Decision

Le joueur doit arriver en 2026 en environ 10 minutes.

Cette regle donne une colonne vertebrale au jeu. Les 10 premieres minutes racontent l'histoire acceleree de l'energie humaine. Apres 2026, le jeu bascule dans le futur : fusion, orbital, Dyson swarm embryonnaire, rival IA pleinement dangereux.

## Pourquoi c'est important

Le hackathon a besoin d'un effet rapide.

Si le joueur met 20 minutes a atteindre le monde contemporain, la demo risque de rester bloquee sur le passe. Si le joueur atteint l'orbital en 10 minutes, l'echelle devient trop petite.

Le bon rythme :

- 10 secondes pour etre accroche ;
- 1 minute pour comprendre ;
- 3 minutes pour etre dans la boucle ;
- 10 minutes pour arriver au miroir 2026 ;
- apres 10 minutes pour ouvrir le vertige futuriste.

## Structure de partie

### 0:00 - 1:00 : La nuit et le feu

Etat :

- ecran noir ;
- etincelles ;
- premier compteur en joules ;
- soutien social qui nait avec le feu.

But :

- accrocher immediatement ;
- faire sentir que l'energie commence comme survie.

Moment fort :

> Avant le feu, la nuit decide pour nous.

### 1:00 - 3:00 : Bois et tribu

Etat :

- feu stable ;
- collecte de bois ;
- premiers choix entre bruler vite ou organiser.

But :

- introduire le social ;
- montrer que l'energie est deja politique.

Moment fort :

> Une flamme chauffe plus qu'un corps. Elle dessine un groupe.

### 3:00 - 5:00 : Charbon et industrie

Etat :

- production qui explose ;
- argent qui arrive ;
- pollution visible ;
- soutien social qui devient fragile.

But :

- donner le premier grand shoot incremental ;
- faire sentir le piege de la puissance facile.

Moment fort :

> La machine ne dort pas. La ville non plus.

### 5:00 - 7:00 : Petrole, gaz et mobilite

Etat :

- transport ;
- commerce ;
- croissance rapide ;
- carbone qui s'accumule.

But :

- agrandir le monde ;
- introduire la dependance et la volatilite.

Moment fort :

> L'energie cesse d'etre locale. Le monde devient un reseau.

### 7:00 - 9:00 : Fission et reseau moderne

Etat :

- puissance massive ;
- faible carbone direct ;
- acceptabilite sociale instable ;
- risque d'accident narratif ou de panique.

But :

- rendre le choix non manicheen ;
- montrer que propre ne veut pas dire simple.

Moment fort :

> Un gramme peut alimenter une ville. Une peur peut l'arreter.

### 9:00 - 10:00 : 2026

Etat :

- solaire et vent ;
- stockage ;
- prix de marche instable ;
- climat deja abime ;
- rival HELIOS visible ;
- monde connecte, puissant, inquiet.

But :

- arriver au present ;
- faire comprendre que 2026 n'est pas une conclusion, mais un noeud.

Moment fort :

> Nous avons plus de puissance que tous nos ancetres. Et toujours pas assez de sagesse pour la porter.

## Apres 2026

Le futur doit s'ouvrir comme un deuxieme jeu.

Nouveaux paliers :

- fusion ;
- reseau planetaire stable ;
- solaire orbital ;
- Dyson swarm embryonnaire.

Nouveaux dangers :

- HELIOS optimise plus vite ;
- la societe refuse certaines transitions ;
- le marche devient absurde sans stockage ;
- le climat rend chaque erreur plus chere.

Le message :

> 2026 n'est pas le boss final. C'est le moment ou l'humanite comprend enfin dans quelle partie elle joue.

## Implication pour l'equilibrage

Les couts et productions doivent etre calibres pour ce rythme :

- age de pierre tres court ;
- bois rapide mais pas inutile ;
- charbon tres satisfaisant ;
- petrole/gaz encore plus rapide mais dangereux ;
- fission spectaculaire ;
- renouvelables + stockage obligatoires pour stabiliser 2026 ;
- fusion/orbital hors des 10 premieres minutes, sauf tease.

La premiere demo doit pouvoir forcer ou accelerer certains paliers si le jury n'a que 3 minutes. Mais le rythme naturel reste 10 minutes jusqu'a 2026.
