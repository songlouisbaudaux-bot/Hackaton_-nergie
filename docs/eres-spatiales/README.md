# Ères Spatiales — Vue d'ensemble

Suite de l'ère atomique. Ces ères sont **spéculatives** — elles commencent là où l'histoire s'arrête (2026) et projettent l'humanité dans son futur énergétique.

La philosophie du jeu reste la même : **les sources s'accumulent, ne se remplacent pas.** En atteignant l'énergie du vide, le joueur a encore des feux de camp qui brûlent quelque part.
