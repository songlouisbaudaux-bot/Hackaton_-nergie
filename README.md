# Prometheus Protocol

Hackaton Energie - jeu incremental web.

## Pitch

Tu commences dans le noir, a l'age de pierre, avec un seul geste : faire naitre le feu.

Chaque clic produit un peu d'energie. Chaque source d'energie agrandit le monde. Chaque transition te donne plus de puissance, mais rend la civilisation plus fragile si tu oublies le climat, le marche ou le soutien social.

Le but n'est pas seulement de produire toujours plus. Le but est de franchir l'echelle energetique sans provoquer l'effondrement.

## Direction

Reference de structure : Universal Paperclips.

Difference philosophique : Paperclips pousse une logique absurde jusqu'a la destruction. Prometheus Protocol pose une autre question : et si l'intelligence ne se mesurait pas a la vitesse de production, mais a la capacite de survivre a sa propre puissance ?

## Debut fort

Le jeu ne commence pas par un tableau de bord. Il commence par une mini-cinematique interactive :

1. ecran noir, froid, presque vide ;
2. une phrase : "Avant le feu, la nuit decide pour nous." ;
3. un seul bouton : "Frotter deux pierres" ;
4. une etincelle apparait ;
5. le premier compteur nait : quelques joules, puis quelques Wh ;
6. la chaleur revele les premiers humains autour du feu ;
7. le vrai jeu se deploie progressivement.

Le premier effet "waouh" doit arriver en moins de 10 secondes.

## Scope hackathon

Version jouable visee :

- incremental temps reel ;
- progression age de pierre -> bois -> charbon -> petrole/gaz -> fission -> renouvelables + stockage -> 2026 en 10 minutes ;
- apres 2026 : fusion, orbital, civilisation de type Kardashev ;
- trois jauges principales : energie, argent, soutien social ;
- une contrainte climat/carbone ;
- un rival IA scriptable qui optimise trop vite et ignore le social ;
- cinematique d'ouverture courte, interactive, rejouable ;
- fin claire : atteindre l'orbital sans effondrement.
